# DeskMascot 使い方ガイド

スマホの画面にキャラクターを表示して、お散歩させたり おしゃべりしたりできるアプリです。

― はじめての方向け・やさしい説明書 ―

> このページはGitHub上でそのまま読める版です。印刷したい方向けにPDF版も用意しています。

---

## このアプリについて

このアプリは、スマホのホーム画面や他のアプリの上に、お好きなキャラクターの画像を重ねて表示します。キャラクターは画面の中を歩き回ったり、話しかけると吹き出しで答えてくれたり、なでると喜んでくれたりします。

キャラクターの画像やセリフは自分で自由に設定できます。会話機能を使いたい場合は、無料で取得できる「APIキー」というものが必要です（あとで説明します。使わなくても表示・お散歩・なで遊びは楽しめます）。

> ※ このアプリはAndroid（アンドロイド）専用です。iPhoneでは動きません。

## 最初に読んでください（安全のおはなし）

このアプリは「他のアプリの上に表示する」という特別な機能を使います。これはキャラクターを画面に重ねて出すために必要なものです。

難しく考えなくて大丈夫です。**「大事な操作（銀行アプリやパスワード入力など）のときはキャラをしまう」** とだけ覚えておけば、安心して使えます。

## 準備するもの

- Android（アンドロイド）のスマホ
- 配布された「DeskMascot」のインストール用ファイル（ファイル名が `.apk` で終わるもの）
- キャラクターにしたい画像（なくても仮の姿で動きます。あとから設定できます）
- （会話機能を使う場合だけ）Googleアカウントと「APIキー」 ※後ろの章で説明します

## ステップ1：アプリをインストールする

配布された `.apk` ファイルをスマホに取り込んだら、次の手順で入れます。お使いのスマホによって画面の文言が少し違うことがありますが、流れは同じです。

1. ファイル管理アプリやメッセージなどから、受け取った「DeskMascot.apk」をタップします。
2. 「セキュリティのため、この提供元のアプリはインストールできません」といった案内が出ることがあります。これは初めてのアプリで普通に出るものなので、あわてなくて大丈夫です。「設定」をタップします。
3. 「この提供元を許可」または「不明なアプリのインストールを許可」というスイッチが出るので、オンにします。
4. 前の画面に戻ると、インストールが続けられます。「インストール」をタップします。
5. 「インストールが完了しました」と出たら成功です。「開く」または、ホーム画面に増えた DeskMascot のアイコンから起動できます。

## ステップ2：キャラクターを作る（最初に必要）

このアプリには最初、キャラクターが入っていません。まず設定画面で、自分のキャラクターを作りましょう。**これをしないとキャラクターは表示できません。**

1. DeskMascotを開いて、「設定」ボタンを押します。
2. 「プリセット名（キャラ名）」に好きな名前を入れます。
3. 「画像設定」で、少なくとも「正面」の画像を選びます（タップするとスマホの写真から選べます）。余裕があれば右向き・左向き・後ろ姿なども入れると、歩く姿がより可愛くなります。
4. 必要なら、大きさ・歩く速さ・セリフなども設定します（あとから変えられます）。
5. いちばん下の「保存」を押します。これでキャラクターが登録されました。

## ステップ3：表示の許可をして、キャラクターを出す

キャラクターを作ったら、画面に表示するための許可をします。初めてのときだけ必要な作業です。アプリが案内してくれるので、その通りに進めればOKです。

1. 設定画面の下の「保存して表示」、またはホーム画面の「キャラクターの表示」を押します。
2. 「他のアプリの上に表示」の許可が必要です、という説明が出たら「設定を開く」を押します。
3. 設定の画面で一覧から「DeskMascot」を探し、スイッチをオンにします。
4. スマホの「戻る」でアプリに戻り、もう一度「キャラクターの表示」を押すと、キャラクターが画面に出てきます。

> （会話・通知・カレンダーなどの機能を使うときは、その都度それぞれの許可を聞かれます。使いたい機能のときだけ許可すれば大丈夫です。特に「通知連携」を使う場合は、この後の専用の章を先にお読みください。）

## ステップ4：基本の使い方

### ホーム画面の3つのボタン

- **キャラクターの表示** … キャラクターを画面に出します。
- **設定** … キャラクターの画像・名前・セリフなどを設定します（次の章）。
- **キャラクターを隠す** … キャラクターを画面から消します。大切な操作の前はこれでしまってください。

### キャラクターを出したあとの操作

- **ドラッグ（押したまま動かす）** … キャラクターを好きな場所へ移動できます。
- **長押し** … キャラクターのそばにメニューが出ます（会話・お触りモード・設定・戻る・終了、天気を設定していれば天気も）。
- **お触りモード** … キャラクターを大きく表示して、タップやなでなでで反応してくれます。下の「終了」で戻ります。
- **会話** … 文字で話しかけると、キャラクターが吹き出しで返事します（APIキーの設定が必要）。

## ステップ5：もっとカスタマイズする（任意）

ステップ2で基本のキャラクターは作りましたが、設定画面では他にもいろいろ調整できます。変更したら、いちばん下の「保存」または「保存して表示」を押すと反映されます。

### おもな設定項目

- **ユーザー名** … キャラクターがあなたを呼ぶときの名前です。
- **画像** … 正面・右向き・左向き・後ろ姿・喜怒哀楽など。歩く絵はGIFやアニメーション画像も使えます。間違えたら「削除」で消せます。
- **大きさ・歩く速さ** … スライダーで調整できます。
- **時間帯ごとのセリフ** … 朝・昼・午後・夜・深夜それぞれのあいさつを設定できます。
- **ランダムメッセージ** … ときどき言うセリフです。
- **お触り・拗ね・寝起きのセリフ** … なでたときや、ご機嫌ナナメのときのセリフを設定できます。
- **衣装替え** … 時間帯ごとに衣装を変えられます（例：夜はパジャマ）。
- **不機嫌になる時間帯** … この時間帯になでると、構われたくない様子で好感度が下がります。夜型の方は深夜を外して別の時間帯にするなど、自分の生活に合わせて変えられます。
- **天気・カレンダー連携** … 今日の天気や予定をキャラクターが教えてくれます。※カレンダーは、スマホ標準のカレンダー（Googleカレンダーなど）の予定が対象です。TimeTree（タイムツリー）など別アプリ内だけの予定は取得できません。
- **プリセット** … キャラクター設定を何種類も保存して、切り替えられます。
- **キャラの共有** … 作ったキャラを1ファイルにまとめて、友達に渡せます。APIキーや天気の地点などの個人情報は、共有ファイルには含まれません。

## （使いたい人だけ）会話機能とAPIキーの準備

キャラクターと文字でおしゃべりするには「APIキー」という鍵のような文字列が必要です。GoogleのAIを無料で使えます。少し手間ですが、一度だけの作業です。

### APIキーの取り方（Gemini / 無料）

1. パソコンかスマホのブラウザで「Google AI Studio」と検索し、サイトを開きます（aistudio.google.com）。
2. 自分のGoogleアカウントでログインします。
3. 「Get API key」または「APIキーを取得」を押します。
4. 新しいプロジェクトを作る形でAPIキーが発行されます。表示された文字列（英数字の長い列）をコピーします。
5. DeskMascotの設定画面を開き、「ChatGPT/Gemini APIキー」の欄に貼り付けて保存します。

これで、長押しメニューの「会話」から話しかけると、キャラクターが返事してくれます。

## 困ったときは

### キャラクターが表示されない
- ステップ2で設定画面でキャラクターを作り、「保存」したか確認してください（これをしないと表示できません）。
- 「他のアプリの上に表示」の許可がオンになっているか確認してください（ステップ3）。

### 画像のところにピンクの丸が出る
- 画像がうまく読み込めていません。設定でその画像を選び直してみてください。

### 会話でエラーが出る
- APIキーが正しく貼られているか確認してください。
- 「制限」「429」などと出る場合は、無料の回数を使い切っています。少し時間をおくか、翌日に試してください。
- 繰り返し出るときは、Google AI Studioでキーを作り直してみてください。

### 歩いているときに絵がカクつく
- スマホの性能による表示の問題です。設定で大きさや速さを下げると改善することがあります。

### キャラクターを完全に消したい
- ホーム画面の「キャラクターを隠す」、または画面上部の通知の「終了」を押してください。

## （任意）通知連携を使う場合の設定

「他のアプリに通知が来たらキャラクターが教えてくれる」機能（通知連携）を使うには、少し特別な許可が必要です。順番が大切なので、下の通りに進めてください。

1. アプリ情報の画面（右上の3点メニュー）から「制限付き設定を許可」を済ませます。
2. DeskMascotの設定画面で「通知連携」をオンにし、「通知アクセスの設定を開く」を押します。
3. 一覧から「DeskMascot 通知連携」を選び、オンにします。

> 通知連携の許可が済んだら、ステップ3の「他のアプリの上に表示」の設定に進んでください（まだの場合）。

## （上級・任意）ChatGPTで会話したい場合

会話のAIは、無料で使えるGemini（ジェミニ）のほかに、ChatGPTでお馴染みのOpenAI（オープンエーアイ）のAIも使えます。設定画面のAPIキー欄に、ChatGPT用のキー（`sk` で始まる文字列）を貼るだけで、自動でChatGPTに切り替わります。

### ChatGPTのAPIキーを取る手順

Geminiより少し手間がかかります（電話番号の確認やクレジットカードの登録が必要です）。順番に進めれば大丈夫です。

1. ブラウザで「platform.openai.com」を開きます。※ChatGPTを使うchatgpt.comとは別のサイトです。
2. アカウントを作る（またはChatGPTと同じアカウントでログイン）。電話番号の確認を求められたら、案内に従って認証します。
3. 左メニューの「Settings（設定）」→「Billing（支払い）」を開き、クレジットカードを登録します。
4. 「Add credit（クレジットを追加）」から、最低5ドルをチャージします。
5. 左メニューの「API keys」→「Create new secret key（新しいキーを作成）」を押します。
6. `sk` で始まる文字列が表示されます。**この画面でしか表示されない**ので、すぐにコピーします（あとから見られません）。
7. DeskMascotの設定画面の「ChatGPT/Gemini APIキー」欄に貼り付けて保存します。これでChatGPTで会話できます。

## 最後に（安全のおさらい）

大事な操作（銀行・パスワード入力など）のときは、キャラクターをしまってから行いましょう。わからないことがあったら、このアプリをくれた人に気軽に聞いてくださいね。楽しいマスコットライフを！
